## What is this?

Morsa Toolkit is a browser-based Morse code app built for anyone — whether you've never heard of Morse code or you're trying to get faster at it.

No sign up. No install. Just open it and go.

---

## Features

- **Converter** — Type text and instantly see it in Morse code. Or paste Morse and decode it back to text.
- **Audio** — Hear your Morse code played out loud with real beeps and a progress bar.
- **Trainer** — A mini game where you listen to a letter in Morse and guess what it is. Tracks your score, accuracy, and WPM.
- **Reference** — A full A–Z and 0–9 Morse code cheat sheet. Every character has a play button so you can hear it.
- **About** — A fun, easy read on the history of Morse code. Who made it, why SOS doesn't mean what you think, and how a POW blinked a secret message on live TV.

---

## Background

The whole app runs on a soft lavender gradient with frosted glass cards. There are also 110 colourful particles floating in the background powered by Three.js — move your cursor over them (or drag your finger on mobile) and they react and scatter.

---

## How to run it

No build tools. No npm install. Nothing like that.

1. Download or clone this repo
2. Open `index.html` in any browser
3. That's it

```bash
git clone https://github.com/yourusername/morsa-toolkit.git
cd morsa-toolkit
# open index.html in your browser
```

## Built with

| Thing | What it does |
|---|---|
| HTML + CSS | Structure and the lavender glass UI |
| Vanilla JavaScript | All the logic — converting, audio, trainer game, tabs |
| Web Audio API | Generates the Morse beep sounds in real time, no audio files needed |
| Three.js | Powers the interactive particle background |
| Google Fonts (Inter) | Clean readable font throughout |
